## SLIDE 02 — 문제 정의

### 중소 제조기업이 AI를 도입 못하는 진짜 이유

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   중소기업 AI 도입률          1%                                │
│   (대기업 40% vs 중소기업 1%)                                   │
│                                                                 │
│   "AI 도입에 관심은 있다"     82%                               │
│   그러나 실행은 0에 가까운 이유:                                │
│                                                                 │
│     ① 어디서 시작해야 할지 모름        47%                     │
│     ② 투자 대비 효과 판단 어려움       35%                     │
│     ③ 관련 전문가 접근이 어려움        28%                     │
│     ④ 정부 지원사업 정보 부족          21%                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**정부 목표**: 2030년까지 스마트공장 **12,000개** 구축 (현재 약 30% 달성)

**성장하고 있지만, 여전히 절대 소수** (통계청 「기업활동조사」, KIAT 2025 산업기술통계집 표4-7-1/4-7-2 재인용):
- 제조업 AI 활용기업 수: **2017년 48개 → 2023년 263개 (5.5배 증가)**
- 제조업 "생산공정" 목적 AI 활용기업: **2017년 150개 → 2023년 674개 (4.5배 증가)**
- → 관심과 활용 모두 급증 중이지만, 전체 중소 제조기업(약 40만개) 대비 여전히 0.1%대 — "어떻게 시작할지 몰라 못 하는" 기업이 압도적으로 많다는 방증

**현실의 장벽**:
- 컨설팅 비용 = 1회 수백만원 (중소기업이 감당 불가)
- 업종마다 다른 AI 도입 포인트 — 식품/주조/자동차부품이 같을 수 없음
- 정부 지원사업은 있지만 찾기 어렵고, 우리 회사에 맞는지 모름

## SLIDE 03 — 서비스 개요

### Factory AI Navi — 작동 방식

```
[ 기업 입력 ]                [ AI 분석 ]                 [ 결과 제공 ]
  업종 선택                   동종업계 비교                 진단 레포트
  (12개 핵심 업종)     →      (공공데이터 기반)     →       PDF 다운로드
  공정 정보 입력              AI 도입 우선순위 3개          정부지원 매칭
  기업 규모 입력              ROI 자동 계산                 카카오 알림
  (소기업/중기업)             지원사업 자동 매칭
```

**핵심 차별점 3가지**:

| 기존 컨설팅 | Factory AI Navi |
|-----------|---------------|
| 수백만원 / 1~3개월 | 무료 ~ ₩29만/월 / 5분 |
| 컨설턴트 개인 경험 의존 | 공공데이터 수치 기반 근거 제시 |
| 업종 불문 일반론 | 12개 핵심 업종 심화 특화 분석 |
| 지원사업 별도 검색 필요 | 실시간 자동 매칭 |

**12개 핵심 업종 선정 이유**:
- 국내 중소제조업 사업체의 **78%**를 커버하는 업종 선별

- 일반 제조업 6개 (사업체 수·지원사업 수·성공사례 풍부)

```
│                                                                │
```

**시장 근거 자료** (KIAT 2025 산업기술통계집 — 정량 근거로 인용, 진단 로직에는 미사용):

| 출처 | 표 번호 | 내용 |
|-----|--------|------|
| KIAT 통계집 | 표 4-7-1 | 제조업 **인공지능(AI) 활용기업 수** 추이 — 2017년 48개 → 2023년 263개 (5.5배) |
| KIAT 통계집 | 표 4-7-2 | 제조업 AI 활용 목적별 기업 수 — "생산공정" 목적 150개 → 674개 (4.5배) |
| KIAT 통계집 | 표 4-1 | 기업규모별·업종별(대분류) 상품혁신 활동률 |
| KIAT 통계집 | 표 4-4 | 기업규모별·업종별 정부지원제도(조세/자금/인력 등) 활용률 |

> ※ 위 자료는 업종 대분류(10종) 또는 국가 전체 단위 통계로, 우리 서비스의 12개 KSIC 세부업종 × 규모별
> 벤치마크(불량률·AI도입률 등)와는 별개입니다. 진단 로직의 정량 벤치마크 중 가동률은 KICOX 실측치를 반영했고,
> 나머지(불량률·AI도입률·인건비 등)는 아직 추정치입니다. 시장성·문제 정의 근거는 KIAT 통계를 사용해 역할을 분리했습니다.

**분석 신뢰도 보장 방식**:
- 모든 벤치마크 수치 = 공공데이터 출처 명시
- LLM(AI)은 수치 생성이 아닌 해석만 담당 (Hallucination 방지)
- 진단 결과에 데이터 출처 항상 표시

## SLIDE 06 — AI 기술 구조

### 3단계 AI 분석 파이프라인

```
STEP ① 동종업계 벤치마크 분석
  ↓
  내 회사 가동률 vs 동종업계 평균 비교 (KICOX 실측)
  + 불량률·인당생산액·AI도입률은 참고 추정치로 별도 표기
  → "귀사 가동률 68%, 동종평균 72% 대비 4%p 낮음"

STEP ② AI 적용 우선순위 Top3 도출 (Claude AI + RAG)
  ↓
  사용자가 직접 체크한 현장 문제(설비고장/에너지과다 등)를 출발점으로
  36건+ 동종업계 AI 도입 케이스 검색 + 업종별 특화 AI 유형 결합
  → 우선순위 1: 공구마모 예측유지보수 / 기대효과: 다운타임 40% 감소 (실제 사례 근거)

STEP ③ ROI 시뮬레이션 (공공데이터 기반 자동계산)
  ↓
  인건비 × 절감률 + 가동률개선에 따른 생산증대 + 에너지비용 × 절감률
  - 정부지원금 (자동 매칭)
  → 투자회수기간 8.9개월 / 3년 순이익 1.2억

  ※ 불량률 개선분은 금액 계산에서 제외 — 업종별 실측 통계가 없어 금액화하면
  근거 없는 숫자가 되기 때문. 대신 KICOX 실측 가동률을 기준선으로 삼아
  "가동률이 이만큼 개선되면 생산량이 이만큼 는다"는 방식으로 계산한다.
```

**사용 AI 기술**:
- LLM: Claude claude-sonnet-4-6 (Anthropic)
- 에이전트: LangGraph 멀티 에이전트
- 검색: pgvector + BM25 하이브리드 RAG

**Step 2 — AI 진단 결과** (2분 소요)
```
[동종업계 벤치마크]

  에너지 비용: 30% vs 동종 평균 28% → 개선 필요

[AI 적용 우선순위] — 체크한 문제(설비 자주 고장·에너지 비용 과다) 기반
  1순위: 로 온도 AI 최적화 ★★★ (기대효과: 에너지 15~18% 절감, 유사사례 기준)
  2순위: 예측유지보수 설치   ★★  (기대효과: 가동률 10%p 향상, 유사사례 기준)
  3순위: 품질 AI 모니터링   ★   (기대효과: 품질 균일성 향상, 유사사례 기준)

[ROI 시뮬레이션]
  구축 비용: 8,000만원
  정부지원금: -4,000만원 (에너지효율 스마트공장 50% 지원)
  실 자부담: 4,000만원
  연간 절감: 5,360만원 (에너지 3,360 + 가동률개선 1,200 + 인건비 800)
  투자회수: 8.9개월 | 3년 순이익: 1.2억
```

## SLIDE 08 — 기대효과

### 공공데이터 기반 임팩트 추정

**중소 제조기업 기준 (소기업, 종업원 30~50인, 연매출 5~10억)**

```
          AI 도입 전        AI 도입 후(1년 목표)    절감 효과
가동률    68%   ──────────→  80%              +12%p (+18%)
에너지비  매출9% ──────────→  매출7.5%         -1.5%p (연 750만원↓)
불량률    3.2%  ──────────→  1.8%              -1.4%p (-44%, 참고 추정치)
인당생산  3,200만원 ───────→  3,800만원+        +19% (참고 추정치)
```
> ※ 위 수치는 실측 결과가 아닌 서비스 도입 시 기대 목표치임. 가동률·에너지비는 KICOX 실측 벤치마크
> 기준선 위의 목표, 불량률·인당생산은 참고 추정치 기반 목표로 성격이 다름을 명시함.

**산업 전체 임팩트** (서비스 확산 시):

| 지표 | 현황 | 5년 후 목표 |
