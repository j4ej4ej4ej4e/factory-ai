
## SLIDE 03 — 서비스 개요

### Factory AI Navi — 작동 방식

```
[ 기업 입력 ]                [ AI 분석 ]                 [ 결과 제공 ]
  업종 선택                   동종업계 비교                 진단 레포트
  (12개 핵심 업종)     →      (공공데이터 기반)     →       PDF 다운로드
  공정 정보 입력              AI 도입 우선순위 3개          정부지원 매칭
  기업 규모 입력              ROI 자동 계산                 카카오 알림
  (소기업/중기업)             지원사업 자동 매칭
```

**핵심 차별점 3가지**:

| 기존 컨설팅 | Factory AI Navi |
|-----------|---------------|
| 수백만원 / 1~3개월 | 무료 ~ ₩29만/월 / 5분 |
| 컨설턴트 개인 경험 의존 | 공공데이터 수치 기반 근거 제시 |
| 업종 불문 일반론 | 12개 핵심 업종 심화 특화 분석 |
| 지원사업 별도 검색 필요 | 실시간 자동 매칭 |

**12개 핵심 업종 선정 이유**:
- 국내 중소제조업 사업체의 **78%**를 커버하는 업종 선별
- 뿌리업종 6개 (정부 AI 도입 지원 최우선 대상)
- 일반 제조업 6개 (사업체 수·지원사업 수·성공사례 풍부)

## SLIDE 05 — 활용 공공데이터셋

### 데이터로 뒷받침하는 진단 — Hallucination 없는 수치 근거

```
┌────────────────────────────────────────────────────────────────┐
│                     활용 공공데이터                            │
│                                                                │
│  [인당 인건비·근로시간]                                        │
│  출처: KOSIS 사업체노동력조사 (고용노동부)                     │
│  내용: 12개 업종 × 2개 규모 = 24가지 인건비 벤치마크          │
│                                                                │
│  [인당 생산액 (노동생산성)]                                    │
│  출처: 2025년 산업기술통계집 (KIAT)                            │
│  내용: 제조업 업종별·기업규모별 노동생산성 ('22 기준)          │
│                                                                │
│  [로봇 도입 업종별 현황]                                       │
│  출처: 2024년 기준 로봇산업 실태조사 (KIRIA/KAR)              │
│  내용: 적용산업별 제조용 로봇 생산·출하 현황                   │
│                                                                │
│  [정부지원사업 공고]                                           │
│  출처: KEIT, 국가R&D API, 소진공 (실시간 수집)                │
│  내용: 현재 신청 가능한 제조 AI 지원사업 목록                  │
│                                                                │
│  총 활용 데이터: 12개 업종 × 2개 규모 = 24개 벤치마크 세트   │
│                + 36건+ AI 도입 성공/실패 케이스                │
└────────────────────────────────────────────────────────────────┘
```

**분석 신뢰도 보장 방식**:
- 모든 벤치마크 수치 = 공공데이터 출처 명시
- LLM(AI)은 수치 생성이 아닌 해석만 담당 (Hallucination 방지)
- 진단 결과에 데이터 출처 항상 표시

## SLIDE 06 — AI 기술 구조

### 3단계 AI 분석 파이프라인

```
STEP ① 동종업계 벤치마크 분석
  ↓
  내 회사 수치 vs 동종업계 평균 비교
  (불량률 / 가동률 / 인당 생산액 / AI 도입률 / 에너지 비용)
  → "귀사는 금속가공 소기업 중 불량률 하위 35% 구간"

STEP ② AI 적용 우선순위 Top3 도출 (Claude AI + RAG)
  ↓
  36건+ 동종업계 AI 도입 케이스 검색
  + 업종별 특화 AI 파라미터 적용
  → 우선순위 1: 공구마모 예측유지보수 / 기대효과: 다운타임 40% 감소

STEP ③ ROI 시뮬레이션 (공공데이터 기반 자동계산)
  ↓
  인건비 × 절감률 + 불량비용 × 개선률 + 에너지비용 × 절감률
  - 정부지원금 (자동 매칭)
  → 투자회수기간 8.9개월 / 3년 순이익 1.2억
```

**사용 AI 기술**:
- LLM: Claude claude-sonnet-4-6 (Anthropic)
- 에이전트: LangGraph 멀티 에이전트
- 검색: pgvector + BM25 하이브리드 RAG

## SLIDE 07 — 서비스 사용 예시 (데모 시나리오)

### 시나리오: 열처리 업체 A사 (종업원 45인, 소기업)

**Step 1 — 입력** (1분 소요)
```
업종: 열처리 (C302) | 규모: 소기업 (45인)
현재 불량률: 3.0% | 가동률: 68% | 연간 생산액: 4.2억
에너지 비용: 매출의 30% | 설비 노후도: 12년
주요 문제: ☑ 에너지 비용 과다  ☑ 설비 자주 고장
```

**Step 2 — AI 진단 결과** (2분 소요)
```
[동종업계 벤치마크]
  불량률: 3.0% vs 동종 평균 2.5% → 하위 40% 구간
  가동률: 68% vs 동종 평균 72%  → 하위 35% 구간
  에너지 비용: 30% vs 동종 평균 28% → 개선 필요

[AI 적용 우선순위]
  1순위: 로 온도 AI 최적화 ★★★ (기대효과: 에너지 15~18% 절감)
  2순위: 예측유지보수 설치   ★★  (기대효과: 가동률 10%p 향상)
  3순위: 경도 AI 예측 시스템  ★   (기대효과: 불량률 0.8%p 개선)

[ROI 시뮬레이션]
  구축 비용: 8,000만원
  정부지원금: -4,000만원 (에너지효율 스마트공장 50% 지원)
  실 자부담: 4,000만원
  연간 절감: 5,360만원 (에너지 3,360 + 설비 1,200 + 인건비 800)
  투자회수: 8.9개월 | 3년 순이익: 1.2억
```

**Step 3 — 지원사업 매칭 Top3** (자동)
```
[긴급] ① 에너지절감형 스마트공장 구축 지원 (산업부)
         지원금 최대 4,000만원 | 마감: D-3 | 자부담 50%
       ② 뿌리업종 AI 응용상용화 지원 (소진공)
         지원금 최대 3,000만원 | 마감: D-15 | 자부담 40%
       ③ 지역특화 스마트공장 보급 (중기부)
         지원금 최대 2,000만원 | 마감: D-30 | 자부담 50%
```

## SLIDE 08 — 기대효과

### 공공데이터 기반 임팩트 추정

**중소 제조기업 기준 (소기업, 종업원 30~50인, 연매출 5~10억)**

```
          AI 도입 전        AI 도입 후(1년)        절감 효과
불량률    3.2%  ──────────→  1.8%              -1.4%p (-44%)
가동률    68%   ──────────→  80%              +12%p (+18%)
에너지비  매출9% ──────────→  매출7.5%         -1.5%p (연 750만원↓)
인당생산  3,200만원 ───────→  3,800만원+        +19%
```

**산업 전체 임팩트** (서비스 확산 시):

| 지표 | 현황 | 5년 후 목표 |
|-----|-----|-----------|
| 스마트공장 구축 건수 | 약 3,600개 (2024) | 12,000개 (2030 정부 목표) |
| 미활용 정부지원금 규모 | 연 추정 2,000억+ | 매칭률 향상으로 집행률 개선 |
| 중소기업 AI 도입률 | 1% (2024) | 10%+ (2030 기대) |

**사회적 가치**:
- 제조 경쟁력 강화: 뿌리업종 AI 도입으로 단가 경쟁력 5~15% 개선
- 고용 구조 변화: 단순 반복작업 → 기술직 전환 (일자리 질 향상)
- 탄소 감축: 에너지 최적화 AI로 제조업 탄소 배출 5~10% 감소 기여
